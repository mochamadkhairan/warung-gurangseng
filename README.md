# 🍽️ Warung Gurangseng

Website **Warung Gurangseng** adalah sebuah website company profile sederhana yang menampilkan informasi mengenai warung makan, menu yang tersedia, serta gambaran visual dari tempat usaha tersebut. Website ini dibuat sebagai project frontend untuk menampilkan desain website modern, responsif, dan mudah diakses oleh pengunjung.

Project ini juga digunakan sebagai **portfolio project** untuk menunjukkan kemampuan dalam membangun website menggunakan teknologi dasar web.

---

## 🌐 Live Demo

Website ini sudah dideploy dan dapat diakses melalui:

- GitHub Pages  
- Cloudflare Pages
- Vercell

Silakan lihat hasilnya di:
https://mochamadkhairan.github.io/warung-gurangseng/
